# Hospital Management System

A desktop-based Hospital Management System developed using Java Swing, JDBC, and MySQL. The system helps manage patient records, room allocation, employee details, ambulance services, and patient discharge operations.

## Features

- Login System
- Add New Patient
- Patient Information
- Update Patient Details
- Room Management
- Search Room
- Employee Information
- Ambulance Information
- Patient Discharge
- Database Connectivity using JDBC

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- rs2xml.jar

## Database

MySQL is used to store:
- Patient Records
- Employee Details
- Room Information
- Ambulance Information
- Login Credentials

## How to Run

1. Install Java JDK
2. Install MySQL
3. Create the database
4. Import SQL tables
5. Add mysql-connector-j.jar
6. Add rs2xml.jar
7. Run Login.java
